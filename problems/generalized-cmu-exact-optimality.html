Short Statement

For scheduling a multiclass queue with convex delay costs, the generalized cμc\mucμ rule is optimal in heavy traffic; is it (or some clean index policy) exactly optimal outside the heavy-traffic limit?

Open Problem

is the generalized cμc\mucμ rule (or another explicit index policy) exactly optimal for the original (non-asymptotic) stochastic control problem, and if not, characterize the exact optimal policy?

What Is Known

Heavy-traffic (diffusion-scale) asymptotic optimality is established for single flexible server (Van Mieghem 1995) and multi-server (Mandelbaum–Stolyar 2004). For linear costs Ci(x)=cixC_i(x)=c_i xCi​(x)=ci​x the rule reduces to the classical cμc\mucμ rule, which is exactly optimal (Klimov / Cox–Smith). For strictly convex costs, exact optimality is known only in special cases; the general exact-optimal policy is open.

Remaining open problem. is the generalized cμc\mucμ rule (or another explicit index policy) exactly optimal for the original (non-asymptotic) stochastic control problem, and if not, characterize the exact optimal policy?
