Computing EF1 and Pareto-optimal allocations in polynomial time

Short Statement

For indivisible goods with additive valuations, an allocation that is both EF1 (envy-free up to one good) and Pareto-optimal always exists, but the known algorithm is only pseudo-polynomial. Can such an allocation be computed in truly polynomial time?

Open Problem

Determine whether a truly polynomial-time algorithm exists.

Statement & Model

There are nnn agents with additive valuations vi:2M→Z≥0v_i:2^M\to\mathbb{Z}_{\ge0}vi​:2M→Z≥0​ over a set MMM of mmm indivisible goods, with the valuations given in binary. An allocation is EF1 (envy-free up to one good) if any agent's envy toward another vanishes after removing some single good from the envied bundle, and (fractionally) Pareto-optimal if no other (fractional) allocation makes some agent better off without making another worse off. Such an allocation always exists (the maximum-Nash-welfare allocation is one), so the question is purely computational: is there an algorithm running in time poly(n,m,log⁡Vmax⁡)\mathrm{poly}(n,m,\log V_{\max})poly(n,m,logVmax​) (where Vmax⁡V_{\max}Vmax​ is the largest value) that outputs an allocation that is simultaneously EF1 and fractionally Pareto-optimal? The known algorithm runs in only pseudo-polynomial time (polynomial in the magnitude of values, not their bit-length). The problem is in PLS and is not known to be PLS-hard, NP-hard, or in P. Determine whether a truly polynomial-time algorithm exists.

What Is Known

Upper bound: pseudo-polynomial (Barman-Krishnamurthy-Vaish 2018); truly polynomial for a constant number of agents or a constant number of distinct values. No lower bound known (not known NP-hard or PLS-hard). Listed as an open problem in the Aziz-Li-Moulin-Wu survey.

Remaining open problem. Determine whether a truly polynomial-time algorithm exists.
