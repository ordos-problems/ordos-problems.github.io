A polynomial pivot rule for the simplex method

Short Statement

Is there a pivot rule under which the simplex method always solves a linear program in a number of pivot steps bounded by a polynomial in the number of variables and constraints?

Open Problem

does there exist a (deterministic or randomized) pivot rule for which the number of pivot steps is bounded by a polynomial in mmm and nnn for every linear program?

Statement & Model

The simplex method solves max⁡{c⊤x:Ax≤b, x≥0}\max\{c^\top x : Ax\le b,\ x\ge 0\}max{c⊤x:Ax≤b, x≥0} by walking along edges of the feasible polyhedron from vertex to adjacent improving vertex; a \emph{pivot rule} chooses, at each vertex, which improving edge to follow. Klee and Minty (1972) showed Dantzig's most-negative-reduced-cost rule can take 2n−12^n-12n−1 steps on a deformed cube, and analogous exponential lower bounds are known for essentially every classical deterministic rule (Bland, steepest edge, greatest improvement, etc.). Question: does there exist a (deterministic or randomized) pivot rule for which the number of pivot steps is bounded by a polynomial in mmm and nnn for every linear program?

What Is Known

Best known are subexponential randomized rules: Random-Facet achieves exp⁡(O(nlog⁡n))\exp(O(\sqrt{n\log n}))exp(O(nlogn​)) pivots (Kalai 1992; Matoušek–Sharir–Welzl), later shown essentially tight for such rules (Friedmann–Hansen–Zwick). No polynomial pivot rule is known. The question is intimately tied to the polynomial Hirsch conjecture (a polynomial diameter bound is necessary for any polynomial pivot rule).

Remaining open problem. does there exist a (deterministic or randomized) pivot rule for which the number of pivot steps is bounded by a polynomial in mmm and nnn for every linear program?
