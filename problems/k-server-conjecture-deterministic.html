Short Statement

With kkk mobile servers in a metric space serving requested points by moving a server to each request, is there for every metric space a deterministic online algorithm whose cost is at most kkk times the optimal offline cost?

Open Problem

Determine whether the deterministic competitive ratio equals kkk for every metric space.

Statement & Model

In the kkk-server problem, an algorithm controls kkk mobile servers located in a metric space MMM and must serve a sequence of requested points: each request must be served by moving some server to the requested point, and the cost is the total distance moved by all servers. The deterministic kkk-server conjecture asks: for every metric space MMM with ∣M∣≥k+1|M|\ge k+1∣M∣≥k+1, does there exist a deterministic online kkk-server algorithm AAA and a constant cMc_McM​ (depending only on MMM) such that costA(σ)≤k⋅OPT(σ)+cM\mathrm{cost}_A(\sigma)\le k\cdot \mathrm{OPT}(\sigma)+c_McostA​(σ)≤k⋅OPT(σ)+cM​ for all request sequences σ\sigmaσ, where OPT(σ)\mathrm{OPT}(\sigma)OPT(σ) is the optimal offline cost? Equivalently, is the deterministic competitive ratio exactly kkk? The matching lower bound of kkk is already known, so the question is whether this lower bound is achievable. The best general upper bound is 2k−12k-12k−1. Determine whether the deterministic competitive ratio equals kkk for every metric space.

What Is Known

The lower bound of kkk is classical, so the conjecture asks whether this bound is always achievable. The best general deterministic upper bound is 2k−12k-12k−1, obtained by Koutsoupias and Papadimitriou through the Work Function Algorithm. The conjectured ratio kkk is known in several special cases, including k=2k=2k=2, line metrics, tree metrics, and metrics with only k+1k+1k+1 or k+2k+2k+2 points. The full metric-space case remains open.

Remaining open problem. Determine whether the deterministic competitive ratio equals kkk for every metric space.
