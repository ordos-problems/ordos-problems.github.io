Short Statement

Color elements of a set system red/blue to minimize the worst set imbalance (discrepancy). If every element lies in at most ttt sets, the Beck-Fiala theorem gives at most 2t−12t-12t−1; the conjecture is that the truth is only O(t)O(\sqrt{t})O(t​), independent of the number of sets or elements.

Open Problem

Prove or refute the Beck-Fiala conjecture that every degree-ttt set system has discrepancy O(t)O(\sqrt t)O(t​).

Statement & Model

Color the elements of a set system red and blue so that every set is as balanced as possible; the discrepancy is the worst imbalance over all sets. Let S\mathcal{S}S be a set system in which every element belongs to at most ttt sets (equivalently, a matrix A∈{0,1}m×nA \in \{0,1\}^{m \times n}A∈{0,1}m×n with at most ttt ones per column). \textbf{Conjecture (Beck-Fiala):} disc(S)=min⁡x∈{±1}n∥Ax∥∞=O(t)\mathrm{disc}(\mathcal{S}) = \min_{x \in \{\pm 1\}^n}\|Ax\|_\infty = O(\sqrt{t})disc(S)=minx∈{±1}n​∥Ax∥∞​=O(t​), independent of mmm and nnn. The Beck-Fiala theorem guarantees discrepancy at most 2t−12t-12t−1.

What Is Known

Beck and Fiala proved the original 2t−12t-12t−1 discrepancy bound, and Bukh later improved this slightly to 2t−O(log⁡∗t)2t-O(\log^* t)2t−O(log∗t). A different line of work based on Banaszczyk's vector-balancing theorem gives O(tlog⁡n)O(\sqrt{t\log n})O(tlogn​), which has the conjectured dependence on ttt but still depends on the number of elements. Bansal and Jiang prove the conjectured O(t)O(\sqrt{t})O(t​) bound in the regime t≥log⁡2nt\ge \log^2 nt≥log2n. The remaining open case is the full, dimension-free O(t)O(\sqrt{t})O(t​) bound for every ttt.

Remaining open problem. Prove or refute the Beck-Fiala conjecture that every degree-ttt set system has discrepancy O(t)O(\sqrt t)O(t​).
