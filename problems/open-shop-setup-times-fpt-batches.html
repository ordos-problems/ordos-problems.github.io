Short Statement

Is unit-time open-shop makespan scheduling with sequence-dependent batch setup times fixed-parameter tractable in the number of batches?

Open Problem

Determine whether batch/open-shop scheduling with setup times is fixed-parameter tractable in the number of batches alone.

Statement & Model

In O∣pij=1,STsd,b∣Cmax⁡O|p_{ij}=1,ST_{sd,b}|C_{\max}O∣pij​=1,STsd,b​∣Cmax​, each job has one unit-time operation on each machine, jobs are partitioned into ggg batches, and a machine incurs a sequence-dependent setup time spqs_{pq}spq​ when it switches from processing a job of batch ppp to a job of batch qqq. The processing order of a job's operations is unconstrained, each machine handles at most one operation at a time, and each job is processed by at most one machine at a time. The objective is to minimize makespan. The open problem asks whether this problem is fixed-parameter tractable parameterized by the number of batches ggg, ideally also under high-multiplicity encoding where the number of jobs in each batch is given compactly.

What Is Known

The problem generalizes the traveling salesperson problem and is NP-hard already for one machine. Van Bevern and Pyatkin give algorithms for cases where every batch contains more jobs than machines, and polynomial algorithms when both the number of machines and number of batches are constants. Mnich and van Bevern list FPT in the number of batches alone as Open Problem 14.

Remaining open problem. Determine whether batch/open-shop scheduling with setup times is fixed-parameter tractable in the number of batches alone.
