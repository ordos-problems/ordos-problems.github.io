Short Statement

For indivisible chores (which impose costs), an EFX allocation is one where any envy agent iii feels toward jjj vanishes after dropping any single chore from iii's own bundle. Does an EFX allocation always exist for additive chores with three agents?

Open Problem

does an EFX allocation always exist for chores with additive costs and n=3n=3n=3 agents?

Statement & Model

Let N={1,…,n}N=\{1,\dots,n\}N={1,…,n} be a set of agents and MMM a set of indivisible chores, with each agent iii having an additive cost function cic_ici​ (chores impose costs / disutilities). In words, any envy agent iii feels toward agent jjj vanishes upon dropping any single (positively costly) chore from iii's own bundle. Unlike the goods setting, the situation for chores is harsher: with additive costs and four or more agents, EFX allocations can provably fail to exist. Question: does an EFX allocation always exist for chores with additive costs and n=3n=3n=3 agents?

What Is Known

For superadditive cost functions, EFX allocations can fail to exist even with three agents. For additive costs, nonexistence examples are known for some number of agents n≥4n\ge 4n≥4, so the general chores analogue of the goods conjecture is false. The remaining small case of additive chores with exactly three agents is still open. Positive results are known when the number of chores is at most n+2n+2n+2, for identical-ordering costs, and for bi-valued costs; in general, a 222-EFX allocation always exists for additive chores.

Remaining open problem. does an EFX allocation always exist for chores with additive costs and n=3n=3n=3 agents?
